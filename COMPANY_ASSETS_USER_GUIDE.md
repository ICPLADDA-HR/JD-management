# Company Assets - คู่มือการใช้งาน

## ภาพรวม
ระบบ Company Assets ช่วยให้คุณสามารถกำหนดทรัพย์สินบริษัทที่จะมอบให้กับแต่ละตำแหน่งงานได้ โดยรองรับทั้งทรัพย์สินที่ตั้งค่าไว้ล่วงหน้าและทรัพย์สินที่เพิ่มเองได้

## การตั้งค่าทรัพย์สินบริษัท

### 1. เข้าสู่หน้า Settings
- คลิกที่เมนู **Settings**
- เลือก **Company Assets**

### 2. จัดการทรัพย์สินบริษัท
- **เพิ่มทรัพย์สิน**: กรอกชื่อและคำอธิบาย แล้วคลิก "Add Asset"
- **แก้ไขทรัพย์สิน**: คลิกปุ่ม Edit ที่ทรัพย์สินที่ต้องการแก้ไข
- **ลบทรัพย์สิน**: คลิกปุ่ม Delete ที่ทรัพย์สินที่ต้องการลบ

### ทรัพย์สินเริ่มต้น
- Laptop (คอมพิวเตอร์แล็ปท็อป)
- Mobile Phone (โทรศัพท์มือถือ)
- Access Card (บัตรผ่านเข้าอาคาร)

## การใช้งานในหน้า Create JD

### ตำแหน่งของ Company Assets
อยู่ระหว่าง **Basic Information** และ **Job Purpose**

### วิธีการเลือกทรัพย์สิน

#### 1. เลือกทรัพย์สินที่ตั้งค่าไว้
- คลิกที่กรอบทรัพย์สินที่ต้องการ
- สามารถเลือกได้หลายรายการ
- กรอบที่เลือกจะเปลี่ยนเป็นสีฟ้า

#### 2. เพิ่มทรัพย์สินเอง (Custom Assets)
- พิมพ์ชื่อทรัพย์สินในช่อง "เพิ่มทรัพย์สินอื่นๆ..."
- คลิกปุ่ม **เพิ่ม** หรือกด Enter
- ทรัพย์สินที่เพิ่มจะปรากฏในรายการพร้อมปุ่มลบ

#### 3. ลบทรัพย์สินที่เพิ่มเอง
- คลิกปุ่มถังขยะ (🗑️) ที่ทรัพย์สินที่ต้องการลบ

### ตัวอย่างการใช้งาน
```
✓ Laptop
✓ Mobile Phone
✓ Company Car (เพิ่มเอง)
✓ Parking Space (เพิ่มเอง)
```

## การใช้งานในหน้า Edit JD

### การแก้ไขทรัพย์สิน
- ระบบจะโหลดทรัพย์สินที่เลือกไว้แล้วมาแสดง
- สามารถเพิ่ม/ลบทรัพย์สินได้เหมือนหน้า Create JD
- ทรัพย์สินที่เพิ่มเองจะแสดงพร้อมปุ่มลบ

### การบันทึก
- คลิกปุ่ม **Save Changes** เพื่อบันทึกการเปลี่ยนแปลง
- ทรัพย์สินทั้งหมดจะถูกบันทึกลงฐานข้อมูล

## การแสดงผลในหน้า View JD

### ตำแหน่งการแสดงผล
อยู่หลังจาก **Basic Information**

### รูปแบบการแสดงผล
- แสดงเป็นกริด 2/3/4 คอลัมน์ (มือถือ/แท็บเล็ต/คอมพิวเตอร์)
- แต่ละทรัพย์สินแสดงในกรอบสีขาวพร้อมเครื่องหมายถูก (✓)
- แสดงทั้งทรัพย์สินที่ตั้งค่าไว้และทรัพย์สินที่เพิ่มเอง

### ตัวอย่างการแสดงผล
```
Company Assets (ทรัพย์สินบริษัท)
┌─────────────┬─────────────┬─────────────┬─────────────┐
│ ✓ Laptop    │ ✓ Mobile    │ ✓ Company   │ ✓ Parking   │
│             │   Phone     │   Car       │   Space     │
└─────────────┴─────────────┴─────────────┴─────────────┘
```

## การพิมพ์เอกสาร

### รูปแบบการพิมพ์
- ทรัพย์สินจะแสดงในเอกสารที่พิมพ์
- จัดเรียงเป็นกริดที่อ่านง่าย
- รวมอยู่ในส่วน Basic Information

## คำถามที่พบบ่อย (FAQ)

### Q: ทรัพย์สินที่เพิ่มเองจะหายไหมถ้าลบทรัพย์สินในหน้า Settings?
A: ไม่หาย ทรัพย์สินที่เพิ่มเองจะถูกบันทึกเป็นข้อความในฐานข้อมูล ไม่เกี่ยวข้องกับรายการใน Settings

### Q: สามารถเพิ่มทรัพย์สินเองได้กี่รายการ?
A: ไม่จำกัดจำนวน แต่ควรเพิ่มเฉพาะที่จำเป็นเพื่อความเป็นระเบียบ

### Q: ทรัพย์สินที่เพิ่มเองจะแสดงในรายการ Settings หรือไม่?
A: ไม่แสดง ทรัพย์สินที่เพิ่มเองจะแสดงเฉพาะใน JD นั้นๆ เท่านั้น

### Q: สามารถแก้ไขชื่อทรัพย์สินที่เพิ่มเองได้หรือไม่?
A: ต้องลบแล้วเพิ่มใหม่ (จะมีฟีเจอร์แก้ไขในอนาคต)

### Q: ถ้าไม่เลือกทรัพย์สินเลยจะเป็นอย่างไร?
A: ไม่มีปัญหา ส่วน Company Assets จะไม่แสดงในหน้า View JD

## เคล็ดลับการใช้งาน

1. **ตั้งค่าทรัพย์สินที่ใช้บ่อยใน Settings** เพื่อให้เลือกได้ง่ายและรวดเร็ว
2. **ใช้ Custom Assets สำหรับทรัพย์สินพิเศษ** ที่ใช้เฉพาะบางตำแหน่ง
3. **ตรวจสอบทรัพย์สินก่อนบันทึก** เพื่อความถูกต้อง
4. **ใช้ชื่อที่ชัดเจน** สำหรับทรัพย์สินที่เพิ่มเอง
